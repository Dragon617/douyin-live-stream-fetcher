### 登录方式改造完成
将抖音登录从 Puppeteer启动浏览器 改为 **用户本地浏览器手动登录 + Cookie粘贴保存**。

#### 改动文件：
- `server.js`:
  - 删除 `/api/login/start` 端点（废弃的Puppeteer登录）
  - 新增 `POST /api/login/save-cookies` 端点（接收手动粘贴的Cookie字符串并解析保存）
  - 删除废弃变量：`loginBrowser`、`isLoggingIn`、`browserInstance`、`getBrowser()`
- `public/index.html`:
  - 登录按钮改为调用 `openLoginPage()` - 用 `window.open()` 打开本地浏览器到抖音官网
  - 新增Cookie粘贴弹窗（带4步操作指引 + 复制代码按钮）
  - 新增 `openLoginPage()`、`closeCookieModal()`、`copyCookieCode()`、`saveManualCookies()` 函数
  - 删除废弃函数 `startLogin()` 和变量 `loginCheckTimer`
  - 更新使用说明

#### 新登录流程：
1. 点击「登录抖音」 - 浏览器新标签打开抖音官网
2. 在打开的页面完成扫码/账号登录
3. 按 F12 - Application - Cookies - https://www.douyin.com
4. 双击 Cookie 值复制，或控制台输入 `document.cookie`
5. 回到工具页粘贴 Cookie - 保存
6. 只需保存一次，后续自动使用
